Raspberry Pico

The Pico is unlike the other Pi's. It is a microcontroller, and it runs whatever code you write for it! This makes it really useful for electronics projects. It runs either Python or C/C++.



Frequently Asked Questions

Do I need Pi OS?

No. The Pi runs your code. You'll only need Pi OS for the Zero, 4 and 5.


How do I write code for it?

Most people write Python for their Picos, and the reccomended IDE is Thonny. It also works on Zero, 4 and 5 so you can edit code for your Pico from your Pi!


What are some project examples?

Some examples of what you can do with a Pico include: a robot, lights systems, connecting a display and making a game for it, and a weather sensor. The possibilities are endless.
