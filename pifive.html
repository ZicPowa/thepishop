Frequently Asked Questions

What OS does it run?

The Pi 5 runs a new, updated version of Pi OS, based on Debian Linux. The current version is known as 'Pi OS Bookworm'.


Will older versions of Pi OS work on the Pi 5?

Unfortunately, you'll have to download the latest version of the OS for use on your Pi 5.


What can it be used for?

The Pi 5 is a general purpose computer. You can run practically anything, such as Visual Studio and Python, as long as it is compatible with Linux.
