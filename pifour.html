Frequently Asked Questions

What can I use it for?

The Pi Four will be very useful for projects that need a full-blown computer, rather than a more affordable microcontroller such as the Pico. However, the Pi 5 does have about 2.5x more processing power.


How is it different from the Pico?

Raspberry Pi Zero, 4 and 5 are full computers running an operating system called Linux. The Pico is a microcontroller, so it can only run the programs that you write for it.


What can it do?

You can run heaps of Linux programs on it! These range from VS Code, all the way to games. It can also be used in your DIY projects.
